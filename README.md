# Regex Playground

Interactive regex testing web app. Write patterns, test against strings, see matches highlighted in real time with full group and capture information.

## Features

- Live regex matching with flag toggles (g, i, m, s, u)
- Match highlighting with group details
- Common pattern library (email, URL, phone, date, IP)
- Safe regex execution with timeout protection

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
